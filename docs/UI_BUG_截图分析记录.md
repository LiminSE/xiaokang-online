## 发现并修复的问题

| 问题 | 影响 | 修复 |
| --- | --- | --- |
| 部分区域右侧任务 CG 与当前区域不一致 | 餐厅、工坊、北部自然区会显示默认或上一张 CG，截图验收容易误判为错图 | 新增 3 张 imagegen 区域任务 CG，并让截图流程按区域切换对应任务 |
| 启动页封面草率 | 首屏左侧是程序化几何小镇，右侧标题仍为英文 `XiaokangOnline` | 使用生成图封面 `boot_cover_v2.png`，并替换为透明卡通「小康online」字标 |
| 地图出现无关图元 | 旧裁切装饰会把树、灯柱、杂物等叠到餐厅、工坊、Livehouse 等地图角落 | 主画面撤掉所有 tile/decor 图像叠层，地图改为整图生成 |
| 部分地图偏糊 | 旧背景为 896x640，页面放大后加上高强度 tile/网格显得发脏 | 画布提升到 1792x1280，9 张地图全部使用整图生成背景 |
| 湖边 v2 背景与旧 NPC 坐标错位 | 新背景干净后，个别角色会落到水面边缘 | 同步调整湖边 NPC、候选刷新点和交互点到栈道/岸边 |
| 小图元本身不稳定 | 旧装饰组不是单一语义图元，而是多个对象拼在同一张图里，例如餐厅装饰含户外树和灯柱、庭院装饰会混入水池和石像，直接叠加容易串场 | 停用所有 tile/decor 图像叠加，9 张地图全部改为整图生成地图 |
| 手机启动页标题溢出 | 390px 宽度下 `XiaokangOnline` 被右侧裁掉 | 移动端品牌区改为纵向排列，并固定移动端标题字号 |
| 手机游戏页地图下方留白过大 | 桌面地图最小高度套到移动端，导致地图下方出现空白 | 移动端取消地图容器最小高度 |
| 手机方向按钮遮挡交互提示 | 移动方向键压住底部提示文字 | 移动端交互提示收窄并避让方向键 |
